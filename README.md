# cruise-ai

Self-driven AI developer coaching. Zero effort, maximum signal.

`cruise-ai` observes how you use AI coding tools — then coaches you to use them better. It detects inefficiencies, recommends automations, and teaches you patterns you didn't know existed.

**Two modes:**
- **Teach Mode** — explains *why* and guides you step by step
- **Auto Mode** — just does it (generates Skills, Hooks, MCPs, configs)

## What It Does

- 📊 **Analytics** — usage dashboards, cost tracking, productivity scoring
- 🎯 **Token Optimization** — detect waste, suggest compression, recommend models
- 🧩 **Skill Engine** — recommend, generate, revise, and manage reusable Skills
- 🔌 **MCP Discovery** — detect APIs/patterns suited for MCP servers
- 🪝 **Hook Automation** — detect repetitive commands → generate hooks
- 🧪 **Eval Harnesses** — detect evaluation opportunities, generate test harnesses
- 🧠 **Project Memory** — detect repeated context → generate architecture docs
- 📁 **Repo Intelligence** — analyze repos, detect missing AI configs, generate docs
- 🎓 **Learning Engine** — "Teach Me" tutorials, "Why This?", progressive AI learning paths

## Install

```bash
pip install cruise-ai
```

## Quick Start

```bash
cruise-ai assess          # Profile your AI usage
cruise-ai dashboard       # Open the analytics dashboard
cruise-ai recommend       # Get personalized recommendations
cruise-ai teach           # Interactive learning mode
```

## How It Works

cruise-ai scans your local AI tool sessions (Kiro, Claude Code, Cursor, Codex, and more) to understand your patterns — then applies evidence-based rules to suggest improvements. Everything is computed entirely from local files. No data leaves your machine — your prompts, responses, and code never reach cruise_ai servers.
