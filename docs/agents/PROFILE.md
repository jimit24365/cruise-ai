# Surface spec: the Profile (agent-readable)

A structured reference for AI agents (and humans) on what the **profile**
is, what it contains, and how to read it. Source of truth for the JSON
itself is [`SCHEMA.md`](../../cruise_ai/docs/SCHEMA.md); this is the
surface-level map.

- **What it is:** the credential dashboard — the card you'd show someone.
- **Route:** `/profile` (served by `hub.py` on localhost).
- **Rendered from:** `profile.json` (the ONE assessment JSON). The report
  renders from the same file — one fact never differs between views.
- **Generated by:** `assess` (arithmetic over local signals). Narrative
  text, if present, comes from the opt-in `enrich` step and never changes
  a number.

## Structure (tabs, all from one JSON)

| Tab | Contains | Notes |
|---|---|---|
| **Overview** | builder card (kind, composite, confidence), positioning chips, "Your numbers" stat cards | the at-a-glance identity |
| **Work** | projects with languages, per-repo detail | hidden on shared/redacted JSON |
| **Lab** | experimental, labeled, **never-shareable** readings (e.g. solo-equivalent leverage) + growth edges | excluded from export/publish |
| **Provenance** | every source that fed the assessment + fidelity (deep/counts/presence), what was NOT collected, raw scan link | how it's kept honest |
| **Share** | per-item visibility toggles, redacted-JSON preview, downloads, export/publish commands | the only path off-machine |

## Key JSON fields an agent reads

- `composite` (int|null) · `assessment.confidence` (0–100) ·
  `assessment.sourcesUsed`, `assessment.sessions`, `assessment.dateRange`
- `dimensions{}` — six dimensions, each `{score: int|null, evidence: []}`.
  `score: null` means **insufficient** → render as a dash, never 0.
- `primaryTitle{id,name}`, `archetypes[]`, `workMode`
- `positioning{buildDomain{primary,distribution,evidence},
  leverageMode{current,evidence}, techDomains[]}`
- `leverage` (Lab; not in shareable), `experimental` (not in shareable)
- `enrichment{}` — six narrative blocks (opt-in; words only)

## Invariants

- Null score → "insufficient" (dash), never 0 or an estimate.
- No ranking language anywhere (no percentiles/cohorts).
- Lab + experimental are structurally excluded from the shareable build
  (`schema.build_shareable_profile` + `visibility.py`).
- The avatar image stays in the browser's localStorage — never in the JSON.
